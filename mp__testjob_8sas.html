Detailed Description

Executes a series of procs and data steps to enable performance testing of arbitrary jobs. 

%mp_testjob(
   duration=60*5
)
Parameters
[in]duration=the time in seconds which the job should run for. Actual time may vary, as the check is done in between steps. Default = 30 (seconds).
